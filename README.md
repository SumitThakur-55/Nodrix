# Nodrix

**Nodrix** is a self-hostable visual workflow automation platform. It allows users to design workflows by dragging nodes onto a canvas, connecting them, and executing automation pipelines in real time.

Built with a modern full-stack architecture, Nodrix supports distributed execution, live monitoring, and secure integration with external systems.

---

## Features

- Visual drag-and-drop workflow builder  
- Real-time execution updates via WebSockets  
- Background job processing with Redis queues  
- Trigger-based and scheduled automation  
- Conditional logic, branching, and data transformation  
- Secure credential storage with encryption  

---

## What you can build

- AI automation pipelines  
- API and database integrations  
- Scheduled background workflows  
- Event-driven automation systems
